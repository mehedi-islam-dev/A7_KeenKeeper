## 📖 Project Overview
**KeenKeeper** is a modern, responsive friendship management dashboard designed to help users keep track of their meaningful connections. It allows users to view friend details, track interaction goals, log daily interactions (calls, texts, video calls), and visualize their communication habits through an interactive analytics timeline and a donut chart.

---

## ✨ Key Features

### 📊 Friendship Analytics
Visualize your communication habits! Features a responsive **Donut Chart** (built with Recharts) that analyzes and displays the distribution of different interaction types (Call, Text, Video).

### 👥 Interactive Friend Details
View specific friend profiles, check their status (On-Track, Overdue), and log interactions with a single click. Interactions are instantly saved locally and trigger a beautiful toast notification.

### ⏳ Dynamic Timeline
A dedicated timeline page tracking all your interaction history. Easily **filter** past interactions by 'Call', 'Text', or 'Video' to see exactly how you've been keeping in touch.

### 📱 Fully Responsive Layout
Crafted with **Tailwind CSS Grid & Flexbox**, ensuring a pixel-perfect, balanced layout across desktop, tablet, and mobile screens. Includes a creatively designed custom **404 Error Page** for invalid routes.
